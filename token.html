Description

TSL Token is Fungible token created using the Ethereum blockchain, which will govern ecosystem of TSLPs DeFi NFT Metaverse. TSLT is interchangeable with other tokens, where well-known non-fungible tokens(NFTs) are not interchangeable. TSLT will be used to support different gaming transactions, such as acquiring TSL-AND(Metaverse Land), interacting with in game assets, and staking it to participate in the administration of the decentralized autonomous organization (DAO).

FAQ

What is an NFT?

An NFT is a type of digital asset that reflects real-world objects such as artwork, films, music, and in-game goods. They're bought and traded online, often using cryptocurrency, and they're usually created with the same software as many other cryptos. An NFT (non-fungible token) is a non-interchangeable unit of data stored on a blockchain, a form of digital ledger, that can be sold and traded.
